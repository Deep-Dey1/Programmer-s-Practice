1 Simple Calculator
Write a program that takes two numbers and an operator (+, -, *, /) as input and performs the corresponding operation. Use functions for each operation.

2 Even or Odd
Write a function that takes an integer as input and returns whether the number is even or odd.

3 Greatest of Three Numbers
Write a program that takes three numbers as input and determines the largest among them using conditional statements.

4 Factorial of a Number
Write a function that calculates the factorial of a given number using a loop.

5 Prime Number Check
Write a function that checks whether a given number is prime or not.

6 Number Reversal
Write a program that takes an integer input and prints its reverse. Example: 1234 → 4321.

7 Grade Calculator

80-89: B
70-79: C
60-69: D
Below 60: Fail
